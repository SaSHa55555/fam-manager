FAMILY TASKS MANAGER

Добро пожаловать в Family Tasks Manager!

Это приложение поможет вам организовать домашние дела, назначить задачи для каждого члена семьи и отслеживать их выполнение. С помощью Family Tasks Manager вы сможете:

Создавать списки задач для всей семьи Назначать задачи конкретным членам семьи Устанавливать дедлайны для задач Отслеживать прогресс выполнения задач Общаться с членами семьи по поводу задач
